2024 New York Congressional Districts Plan

##Redistricting Data Hub (RDH) Retrieval Date
02/29/2024

##Sources
This dataset was retrieved from the [New York State Legislative Task Force on Demographic Research and Reapportionment](https://latfor.state.ny.us/maps/)

##Processing
The RDH did not modify any of the data.

##Additional Notes
Enclosed in this zip file the shapefile file for New York's Congressional Districts ("CON24_shapefile_Feb_28_2024") and supporting files, a statewide map ("con-nys.pdf"), a New York City map ("con-nyc.pdf"), a Long Island map ("con-li.pdf"), and a block assignment file ("CON24_Feb_28_2024.dbf").
